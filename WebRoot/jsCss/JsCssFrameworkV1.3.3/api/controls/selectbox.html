selectbox弹出选择框

简介

  • 实现文本框选择弹窗效果

依赖

  • jQuery
  • 弹窗效果依赖于asyncbox.js
  • 弹窗样色依赖于asyncbox.css
  • 按钮的样色依赖Bootstrap的gluphion图标，详细使用方法请参考bootstrap官网

包含

     JS文件

  • libs/asyncbox/AsyncBox.v1.4.js
  • libs/framework/controls/corebox.js
  • libs/framework/controls/selectbox.js

     CSS文件

  • libs/framework/css/gluphicon.css
  • libs/asyncbox/skins/ZCMS/asyncbox.css
  • libs/framework/css/form.css

使用方法

     绑定方法

  • 找到需要绑定的input标签，设置标签的class="selectBox",通过以下class的嵌套来绑定样色。
    class嵌套：data-form→panel→body→selectBox
  • 如需要更改seletbox的默认值(点击这里查看默认值)，则不能用通过以上class="selectBox"方法绑定，需要通过以下js绑定相应id或class来实现。

例子：设置window的height和width

               

    

        

            
            
    
            
            
 
        js部分：
                
                $(function () {
                $("#x").selectBox({                 *****绑定id为x的input标签******
                    window: { width: 900,height: 300}  
                                         });
                  })

默认值

属性名 使用 描述 值类型 默认值
window window: {title:"自定义title", width: 1200, height: 200} title：设置弹窗的标题 string "选择"
width：设置弹窗的宽度 number 600
height：设置弹窗的高度 number 400
url url:"链接地址" 设置弹窗链接地址 string undefined
select select: function ( ){ } 设置被确认选择时调用的函数 object
readonly readonly:true 设置文本可编辑性 boolean false（不可编辑）

例子：

           
    

        

            
            

            
            
           
js部分：
            
            $(function () {
                $("#input4").selectBox({          *******绑定id为input4的input标签*******   
                    window: {                  ************设置弹窗属性************
                        title:"test",
                        width: 1200,
                        height: 200
                    },
                    url:"http://www.baidu.com",   *******设置弹窗链接地址*******                    
                    select: function () {
                        alert("123");         ***设置确认选择时调用的方法*****                       
                    },
                    readonly: false           ****设置文本可编辑性为无****
                });
            })

属性

属性 描述 值类型 默认值
data-url 设置弹窗链接地址 string undefined
title 设置弹窗标题 string "选择"
data-window-width 设置弹窗宽度 string 600
data-window-height 设置弹窗高度 string 400
readonly 设置文本可编辑性 boolean false（不可编辑）
