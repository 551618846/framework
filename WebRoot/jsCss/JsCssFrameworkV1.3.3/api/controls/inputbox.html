inputbox文本框

简介

  • 实现文本框效果

依赖

  • jQuery
  • 文本框样色依赖form.css
  • 输入效果样色依赖core.css

包含

     JS文件

  • libs/framework/controls/inputbox.js

     CSS文件

  • libs/framework/css/core.css
  • libs/framework/css/form.css

使用方法

     绑定方法

  • 找到需要绑定的input标签，设置标签的class="input",通过以下class的嵌套来绑定样色。
    class嵌套：data-form→panel→body→input
           
    

        

            
            
 
  <form class="data-form">
    <div class="panel">
      <div class="body">        
          <input type="text" class="input" />        
      </div>
    </div>
  </form>
