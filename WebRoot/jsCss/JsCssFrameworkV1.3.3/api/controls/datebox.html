datebox日期选择框

简介

  • 实现文本框日期选择效果

依赖

  • jQuery
  • 日期选择框效果和样色依赖DatePicker，详细使用方法请参考WdatePicker帮助文档
  • 按钮的样色依赖Bootstrap的gluphion图标，详细使用方法请参考bootstrap官网

包含

     JS文件

  • libs/DatePicker/WdatePicker.js
  • libs/framework/controls/corebox.js
  • libs/framework/controls/datebox.js

     CSS文件

  • libs/framework/css/gluphicon.css
  • libs/DatePicker/skin/default/datepicker.css
  • libs/framework/css/form.css
  • libs/framework/css/core.css

使用方法

     绑定方法

  • 找到需要绑定的input标签，设置标签的class="dateBox",通过以下class的嵌套来绑定样色。
     class嵌套：data-form→panel→body→dateBox
  • 如需要更改datebox的默认值(点击这里查看默认值)，则不能用通过以上方法绑定，
    需要通过以下js绑定相应id或class来实现。

例子：修改日期显示格式

               

    

        
            
            
            
    
            
            
 
        //js部分：
                
 $(function () {
   $("#Text1").dateBox({                //绑定id为Text1的input标签
      dateFmt: 'yyyy/MM/dd ', //修改了日期显示格式，详细方法请参考WdatePicker的说明文件                 
                    });
            })

       //html部分：

默认值

属性名 使用 描述 值类型 默认值
dateFmt dateFmt: 'yyyy/MM/dd' 设置日期显示格式 string yyyy-MM-dd 00:00:00

属性

属性 描述 值类型 默认值 使用
data-dateFmt 设置日期显示格式 string yyyy/MM/dd data-dateFmt="yyyy-MM-dd"
data-setting 设置日期显示格式 json {dateFmt: 'yyyy/MM/dd '} data-dateFmt="{dateFmt: 'yyyy/MM/dd '}"
