dropdownbox下拉选择框

简介

  • 实现文本框下拉选择效果
  • 分为普通下拉选择框和树形下拉选择框两种

依赖

  • jQuery
  • 下拉列表的效果和样色依赖ztree
  • 按钮的样色依赖Bootstrap的gluphion图标，详细使用方法请参考bootstrap官网

包含

     JS文件

  • libs/zTree_v3/js/jquery.ztree.all-3.5.min.js
  • libs/framework/controls/core.js
  • libs/framework/controls/corebox.js
  • libs/framework/controls/dropdownbox.js

     CSS文件

  • libs/framework/css/gluphicon.css
  • libs/zTree_v3/css/zTreeStyle/zTreeStyle.css
  • libs/framework/css/form.css

使用方法

     绑定方法

  • 普通下拉选择框:找到需要绑定的input标签，设置标签的class="listBox"即可
  • 树形下拉选择框:找到需要绑定的input标签，设置标签的class="treeBox"即可

代码：
  • 上述方法只是绑定了相应的dropdownbox的下拉窗功能，如需绑定完整样色，则通过以下class的嵌套来绑定样色。
     class嵌套：
       普通下拉选择框:data-form→panel→body→listBox
       树形下拉选择框:data-form→panel→body→treeBox
  • 如需要更改dropdownbox的默认值(点击这里查看默认值)，则不能用通过以上方法绑定，
    需要通过以下js绑定相应id或class来实现。

例子：设置为普通下拉多选框

               

    

        
            
            
            
    
            
            
 
        //js部分：
                
$(function () {
     $("#Text1").dropDownBox({             //绑定id为Text1的input标签
         selectType: "list",            //设置为普通下拉框
         multiple: true,             //设置为多选
         onSelect: function () { 
                alert("你触发了选择事件")     //设置选择时触发的事件
                               }  
                });
            })

默认值

属性名 使用 描述 值类型 默认值
url url:"链接地址" 设置下拉框数据链接地址 string undefined
data data:[json] 设置下拉框数据内容 string undefined
selectType selectType: "tree" 设置下拉框列表类型为树形类型 string "tree"
selectType: "list" 设置下拉框列表类型为普通类型
multiple multiple:true 设置下拉选择类型是单选还是多选 boolean false(单选)
onSelect onSelect: function ( ) { } 设置设置选择时触发的事件 obj

属性

属性 描述 值类型 默认值
data-url 设置下拉框数据链接地址 string undefined
data-multiple 设置下拉框类型是多选（true）还是单选（false） boolean false（单选）
data-selectType 设置下拉框类型是普通类型（list）还是树形类型（tree） string tree
data-onSelect 调用选择时触发的事件
data-readonly 设置文本可编辑性 boolean false（不可编辑）
