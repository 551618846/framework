uploadbox文件上传框

简介

  • 实现文件上传框效果

依赖

  • jQuery
  • jquery.form.js
  • 提示弹窗效果依赖于asyncbox
  • 按钮的样色依赖Bootstrap的gluphion图标，详细使用方法请参考bootstrap官网

包含

     JS文件

  • libs/asyncbox/AsyncBox.v1.4.js
  • libs/jquery/jquery.form.js
  • libs/framework/controls/corebox.js
  • libs/framework/controls/core.js
  • libs/framework/controls/uploadbox.js

     CSS文件

  • libs/framework/css/gluphicon.css
  • libs/asyncbox/skins/ZCMS/asyncbox.css
  • libs/framework/css/core.css
  • libs/framework/css/form.css

默认值

属性名 使用 描述 值类型 默认值
name name:'file1' 设置文件上传框的name（当有多个上传框时，name属性不可少） string 'file'
enctype enctype:'' 设置表单数据发送到服务器前的编码方式 string ''
url url:"数据请求地址" 设置数据请求地址 string undefined
autoSubmit autoSubmit:false 设置是否自动提交数据 boolean true
infoPanel infoPanel: {id:"infoPanel1", display: false} id：设置上传信息面板的id string undefined
display：设置上传信息面板的可见性 boolean true（可见）
onSelect onSelect: function ( ){ } 设置被确认选择时调用的函数 object
onDelete onDelete: function ( ){ } 设置数据被确认删除时调用的函数 object
success success: function ( ){ } 设置数据上传成功时调用的函数 object
fail fail: function ( ){ } 设置数据上传失败时调用的函数 object
error error: function ( ){ } 设置数据上传出错时调用的函数 object
params params: { } 设置上传文件url可带参数

属性

属性 描述 值类型 默认值
data-url 设置上传数据url string undefined
data-readonly 设置文本可编辑性 boolean false（不可编辑）
data-infoPanel-id 设置上传信息面板的id string undefined
data-infoPanel-display 设置上传信息面板的可见性 boolean true（可见）
data-onSelect 设置被确认选择时调用的函数 obj
data-onDelete 设置数据被确认删除时调用的函数 obj
data-success 设置数据上传成功时调用的函数 obj
data-fail 设置数据上传失败时调用的函数 obj
data-error 设置数据上传出错时调用的函数 obj
data-params 上传文件url可带参数
